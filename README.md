# DC Print Landing Page

Landing page breve para DC Print con foco en conversión.

## Incluye
- Hero con mensaje claro de impresión CMYK premium.
- Servicios, flujo, galería y contacto sin texto innecesario.
- Botón de WhatsApp fijo/flotante siempre visible.
- Sección de comentarios con campo de texto y calificación por estrellas.
- Fondo del hero con cambio automático de imágenes.

## Ejecutar
```bash
python3 -m http.server 4173
```
Abrir `http://127.0.0.1:4173`.
